# DQ-Time-Step-Generator

Simple form that allows for an easy calculation of the double-quantum evolution times in a cycle-incremented Baum-Pines experiment. Parameter naming follows the Bruker TopSpin convention and [the thesis of C. Franz](https://opendata.uni-halle.de/bitstream/1981185920/8038/1/Dissertation%20Cornelius%20Titus%20Franz.pdf). (large pdf - attention)

Used tech: Vite starter + React/TS + tailwind

Live available at:
https://dq-time-step-generator.vercel.app/
